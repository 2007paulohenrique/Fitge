# Fitge

Fitge é um aplicativo web/mobile dedicado ao suporte a praticantes de musculação para que consigam acompanhar seus treinos e visualizar seus progressos, e aos treinadores para que consigam gerenciar seu trabalho de uma forma eficiente e abrangente.
